新主题开坑（长期更新）

2020-03-30 times read

Contents

    • 未完待续…

Book 这个 hexo-theme 很开放，也很简陋，Home 要自己写，文章列表也要自己引入，相比于我之前用过的主题完全是两个东西。

看了 Hexo 官方文档，感觉应该可以用 js 脚本直接生成，但要学的话这两个月是不可能了。

这个主题不会根据目录生成文章列表，根据排版，列表有两种实现方法。

  • 把文章链接一个个写到到右边的menu中。
  • 直接引用Markdown文件。

考虑到文章标题长度参差不齐（甚至行宽小于标题长度），我就采用了后者。但是Markdown本身扩展性不强，做出来效果不咋地，所以我就考虑了低成本的方法——在Markdown中引用CSS。

自己翻教程写了CSS，桌面端看着还行，但对移动端无适配，看起来很奇怪，就放弃了，凭我现在也写不出什么东西。后来看上了一个开源库 Bulma，翻着文档，踩着坑，最后终于写出了像样的界面（借 gao 鉴 fang 知乎）。

右键插件 -> 打开样式管理器 -> 编写新样式 -> 粘贴下方代码 -> 保存

    background:#ade1f5; /*选中文字背景颜色*/
    color: #000000; /*选中文字颜色*/
}
